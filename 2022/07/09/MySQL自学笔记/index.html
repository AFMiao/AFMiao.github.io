数据库概述以及数据准备

SQL概述

SQL的全称为Structured Query Language, 用于和数据库进行通讯.

SQL是一套标准, 但由于每一个数据库都有自己的特性, 故在使用某个数据库特有的特性时, SQL语句就不是标准的了.

什么是数据库

数据库, 通常是一个或一组文件, 保存了一些符合特定规格的数据. 数据库对应的英语单词是DataBase, 简称DB; 而数据库软件被称作数据库管理系统(DBMS, DataBase Management System), Oracle、SQL Server、MySQL等等均属此列

MySQL概述

MySQL安装

表

表(table)是一种结构化的文件, 可用于储存特定类型的数据. 每个表都有特定的名称, 而且不能重复. 表中有几个概念: 列、行和主键. 其中, 列又被称作字段(column), 行又被称作表中的记录. 每个字段都有字段名称/字段数据类型/字段约束/字段长度.

SQL的分类

  1. 数据查询语言DQL(Data Query Language)
    代表关键字: select
  2. 数据操纵语言DML(Data Manipulation Language)
    代表关键字: insert, delete, update
  3. 数据定义语言DDL(Data Definition Language)
    代表关键字: create, drop, alter
  4. 事务控制语言TCL(Transactional Control Language)
    代表关键字: commit, rollback
  5. 数据控制语言DCL(Data Control Language)
    代表关键字: grant, revoke

导入演示数据

此处展示一下如何利用MySQL命令行客户端来装载数据库.

  1. 连接MySQL: mysql -uroot -p
  2. 创建名为bjpowernode的数据库: create database bjpowernode
  3. 选择数据库: use bjpowernode
  4. 导入数据: source D:\bjpowernode
  5. 删除数据库: drop database bjpowernode

常用命令

查看MySQL版本

  1. mysql --version
  2. mysql -V

创建数据库

  1. 创建一个名为test的数据库: create database test;
  2. 选择名为test的数据库: use test;

查询当前使用的数据库

  1. select database();
  2. 查询数据库版本也可以使用select version();

查询全部字段

有一种很简便的查询全部字段的方法, 即使用select * from emp;命令.

这种方法虽然简单, 但不是很直观. 还是建议在查询全表的时候将全部字段都放进select后. 后面我们用Java连接数据库的时候是需要在Java程序中直接编写SQL语句的. 在这个时候, 如果还使用select *语句就不那么清晰了, 应该将全部字段输入, 以提高可读性.

条件查询

条件查询需要用到where语句. where必须放到from语句表的后面. 支持如下运算符:

运算符 说明
= 等于
<>或!= 不等于
< 小于
<= 小于等于
> 大于
>= 大于等于
between ... and ... 介于两个值之间
is null 判断为空
is not null 不为空
and 并且
or 或者
in 包含, 相当于多个or
not 取非, 主要用于is或in中
like 模糊查询, 支持%或下划线匹配
